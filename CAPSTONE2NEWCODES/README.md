# BasicLevelWebDevCodes / DecisionPhone

A full-stack Flask and Bootstrap application for mobile device decision support, recommendation ranking, comparison, and price trend analytics.

## Run locally

1. Install Python dependencies:
   `py -3 -m pip install -r requirements.txt`
2. Start the Flask app:
   `py -3 run.py`
3. Open `http://127.0.0.1:5000/` in your browser.

## Included

- Flask API with JWT auth, SQLite models, recommendations, trends, reviews, and admin CRUD
- Responsive Bootstrap frontend with Chart.js visualizations
- Seeded phone data in `data/phones.json`
